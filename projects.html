Projects

  • Project on Facial Expression and Race Recognition
    • Trained a facial recognition system from scratch which can recognize the emotion and races by analyzing images/videos.
    Extracted race and face features from training set and trained data according to two strategy, one versus one and one versus all, using four different classifiers which are Support Vector Machine, Training Tree, Logistic Regression and MLP classifier in Python.
    Designed real-time recognition by extracting features from camera with FPS over 10, testing the data in the trained model and giving a recognition of six basic emotional expressions and four human races with accuracy of 80%.
  • Project on Iris Recognition System
    • Designed an iris recognition system in MATLAB using datasets collected by LG2200 as probes and LG4000 as galleries, each of which contains 220 subjects' left and right eye images.
    Handled with invalid data and optimized the model and parameters given in an open source iris recognition software.
    Evaluated the final iris recognition system and get an equal error rate of 8% in ROC and a Rank One Recognition Rate of 90.4% in CMC.
